Kết nối Tracking Sheet ↔ Dashboard ★ Build mới

A GAS Time-trigger (Khuyến nghị)

  • Tracking sheet có cột: Quyền lợi / Deadline / Done (✓)
  • GAS time-trigger chạy mỗi ngày lúc 8:00 sáng
  • Đọc từng tracking sheet → so deadline với ngày hiện tại
  • Nếu deadline qua & chưa done → gửi Telegram alert
  • Cập nhật benefits[] trong GAS DB
  • Dashboard nhận khi bấm Sync GAS hoặc auto khi mở trang
  • Nếu tất cả done → stage tự chuyển Hoàn thành
⭐ Phù hợp nhất — đơn giản, đủ dùng

B Sheet onEdit Trigger (Real-time hơn)

  • Tracking sheet: thêm Apps Script onEdit trigger
  • Mỗi khi user tick ✓ một quyền lợi → trigger kích hoạt ngay
  • GAS cập nhật benefits[] trong DB ngay lập tức
  • Dashboard nhận khi bấm Sync GAS
  • Nếu tất cả done → Telegram notify ngay, stage → Hoàn thành
⚡ Update ngay khi tick done, nhưng cần cấu hình trigger phức tạp hơn và có giới hạn quota Google

Cập nhật UI Dashboard — Hiển thị trạng thái quyền lợi ⚙ Thêm vào dashboard
⚠️ Kanban Card — Badge trễ
Mỗi thẻ Kanban ở giai đoạn Thực thi hiển thị:
• 🔴 Trễ 3 ngày — nếu deadline qua chưa done
• 🟡 Còn 2 ngày — nếu sắp đến hạn
• Progress bar quyền lợi X/N done
📊 Overview — Cần chú ý
Thêm vào phần "Cần chú ý" (đã có):
• Count quyền lợi trễ deadline
• Liệt kê hồ sơ có quyền lợi chưa giao
• Click để mở tracking sheet tương ứng
🏆 Hoàn thành — Auto
Khi tất cả benefits[].done = true:
• Stage tự động chuyển → Hoàn thành
• Telegram: "🏆 CLB X đã hoàn tất tất cả quyền lợi!"
• Dashboard highlight màu xanh lá
📱 Telegram Alert — Sáng hàng ngày
GAS time-trigger 8:00 sáng gửi digest:
• Danh sách quyền lợi trễ hoặc sắp đến hạn
• Số hồ sơ đang thực thi cần theo dõi
• Format ngắn gọn, dễ đọc trên mobile
📋 Tab Thực thi — Nâng cấp
Tab Thực thi trong modal hiển thị:
• Bảng quyền lợi với deadline + trạng thái
• Sync từ tracking sheet tự động
• Link mở trực tiếp Google Sheet tracking
🔔 Thống kê — Kết quả quyền lợi
Trang Thống kê (đã có phần Kết quả quyền lợi):
• Google Maps reviews thu được: X
• Data / contacts: X records
• Tỉ lệ hoàn thành tổng: X%
